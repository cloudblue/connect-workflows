Terminating Status

Description

This subscription status is displayed in case a buyer created a request to cancel a subscription. These requests can be also emulated from Vendor or Distributor portals. Once Vendors approve this cancellation request, a terminating subscription is transferred to the Terminated state. In case Vendors reject this request, the subscription returns to the Active state and the request gets the Failed status.

Prerequisites

A subscription in the Active state and a cancel request.

Transferable statuses

In case a cancel request is apporived, the system assigns the Terminated status to a terminating subscription. If this request is rejected, the system assigns the Active.

Associated transitions

  • Subscription Cancellation
  • Cancellation Approval
