Deploying Request Status

Description

The deploying listing request status is displayed in case a Distributor marked this listing request as deploying. Therefore, Distributors initiate product deployment to a specified marketplace. Otherwise, Distributors reject the deployment operation and this listing request transfers to the Cancelled state.

Listing status

Deploying listing requests are interconnected with the following Listing status: Not Listed

Prerequisites

A listing requests in the Reviewing state.

Transferable statuses

Once a Distributor finishes the product deployment operation, a deploying listing request can be assigned to the Completed status. In case of an error, Distributor can also reject a listing request and assign the Cancelled status to this request.

Associated midway states

  • Reviewing Request Approval
  • Deployment Completion
  • Deployment Cancellation
